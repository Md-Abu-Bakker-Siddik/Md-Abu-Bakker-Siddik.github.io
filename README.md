# MD Abu Bakker Siddik — Portfolio

Single-page portfolio for GitHub Pages.

## Structure

- `index.html` — main site
- `assets/css/main.css` — compiled styles
- `assets/scss/` — SCSS source (edit here, then compile)
- `assets/js/main.js` — navigation, filters, scroll
- `assets/js/themes-data.js` — ThemeForest live demo links

## Build CSS

```bash
npx sass assets/scss/main.scss assets/css/main.css
```

## Legacy URLs

`portfolio.html`, `portfolio1–4.html`, and `contact.html` redirect to sections on `index.html`.
